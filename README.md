# Stop-n-Shop
 E-Commerce Android application 
It covers many material design components, such as: 
- NavigationView  
- CoordinatorLayout  
- AppBarLayout  
- CollapsingToolbarLayout  
- TabLayout  
- FloatingActionButton  
- SwipeRefreshLayout  
- RecyclerView  
- CardView  
and contains features like :
- Animated opening
- Login/Signup
- Google Sign in
- Cart
- Favourites
- Different sections for Men, Women and Kids
