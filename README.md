# Stop-n-Shop
 E-Commerce Android application 
It covers many material design components, such as:**  
- NavigationView  
- CoordinatorLayout  
- AppBarLayout  
- CollapsingToolbarLayout  
- NestedScrollView  
- TabLayout  
- FloatingActionButton  
- SnackBar  
- SwipeRefreshLayout  
- RecyclerView  
- CardView  
- TextInputLayout
